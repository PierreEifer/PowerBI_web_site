Page 1
Map of E‑Fuel Projects
Explore the global project landscape and filter by company, region, and e‑fuel type.

What you see

This page displays an interactive world map showing all E‑Fuel projects included in the dataset. Each point on the map represents a project, and its color indicates the project status (announced, operational, cancelled, etc.).

Available filters
  • Company: Displays projects associated with a specific company.
  • Project: Allows selection of individual projects.
  • Super region: Filters projects by geographic area (Europe, Asia, Africa, etc.).
  • E‑Fuel: Filters by the type of e‑fuel (e‑ammonia, e‑methanol, e‑kerosene, etc.).
How to use this page
  • Click on a geographic area to zoom in.
  • Use the filters on the left and right to focus the analysis on a specific company, e‑fuel type, or region.
  • The total number of projects (shown in the bottom right) updates automatically based on selected filters.

Page 2
Cumulative Production of Chosen Metrics
Track how CO₂ capture, e‑fuel output, and hydrogen production evolve over time.

Main chart

This page visualizes the cumulative production of different metrics over time, such as CO₂ capture, e‑fuel production, and hydrogen (H₂) production in ton/year.

  • The horizontal axis (X‑axis) shows the year.
  • The vertical axis (Y‑axis) shows the cumulative quantity (in tons per year).
  • The bars illustrate how production evolves over time for the selected metric.
Available filters
  • Metric: Choose between CO₂ capture, e‑fuel production, or H₂ production.
  • E‑Fuel: Filters by type of e‑fuel involved.
  • Super region: Filters the data by geographic region.
How to use this page
  • Select the desired metric at the bottom of the page to update the chart.
  • Use the filters to analyze production trends in a specific region.
  • The “Count of projects” indicator automatically updates based on the selected filters.

Page 3
HESC (Liquified Hydrogen Energy Supply Chain)
Follow the pathway from power source to CO₂ capture and e‑fuel output for individual projects.

Overview

This page summarizes the HESC project. It lets you choose one project and see its geographic context view and trace the hydrogen‑to‑e‑fuel pathway from power source through CO₂ capture and fuel output.

Available filter
  • Use the Project slicer on the left to select one project. This controls every other visual on the page.
How to use this page
  • Select the desired project in the project filter.
  • After selecting a project, click the Open Web Link – Click Here + Ctrl button. Hold Ctrl while clicking to open the external reference in your browser.
  • The map on the right shows the project’s location. Zoom/drag to confirm geographical context.
  • Power Source block shows the renewable feed (e.g., Solar).
  • Electrolyze section lists technology, vendor, installed/published/estimated capacities. Blank fields indicate missing published data; estimated values (e.g., 307 MW) provide the model’s best guess.
  • Hydrogen production boxes display published vs. estimated tonnage per year. Blank fields indicate missing published data.
  • CO₂ section captures emissions data: technology, emitter, source type capture company, and captured CO₂ volumes.
  • E‑fx pathway & e‑fuel production show the synthetic fuel route plus published/estimated fuel tonnage per year. Use these to assess project scale and compare them to other entries.
Data Consistency Watch

For some projects, the published values for H₂, CO₂, and E‑Fuel are not consistent with each other.

Estimations are made using stoichiometric coefficients for each type of fuel: ammonia, methanol, methane, kerosene…

This explains the significant discrepancies between the estimate and the published values.

Page 4
Sankey – Projects flow by region, power source, e‑fuel and sector
Visualize how projects move from super region to power source, E‑Fuel type, and end‑use sector in a single flow diagram.

What you see

This page shows a Sankey diagram that connects four stages: Super region, Power source, E‑Fuel type, and End‑use sector. The thickness of each band represents the number of projects flowing between categories, making it easy to see which combinations dominate the portfolio.

Available filters
  • Super region: Focus on specific geographic areas (e.g., Europe, Asia, Africa).
  • Power source: Filter by technology such as wind, solar, hydro, renewable energy or unknown.
  • E‑Fuel type: Limit the view to e‑ammonia, e‑methanol, e‑kerosene, other e‑fuels, etc.
  • End‑use sector: Narrow down to sectors like chemicals, shipping, aviation, mobility, fertilizer and others.
How to use this page
  • Use the filter cards at the bottom (Super region, Power source, E‑Fuel type, End‑use sector) to include or exclude categories.
  • Hover over a band in the Sankey diagram to see the number of projects and percentage for a specific flow.
  • Combine filters across all four dimensions to understand which power sources and e‑fuels dominate in each region and sector.
  • Reset filters to return to the global overview when you want to compare all flows again.

Page 5
Company details
Drill down to individual companies, see where their projects are located, and review key project information.

What you see

This page combines multiple filter panels, a world map and a detailed table. You can filter by Company, CO₂ capture company and other attributes, see project locations on the map, and review a list of projects with hyperlinks to external sources.

Available filters
  • Company: Filter projects by the main company involved.
  • CO₂ capture company: Focus on projects associated with specific capture technology providers.
  • Additional slicers (e.g., country or region) may be available depending on the report configuration.
How to use this page
  • Use the Company and CO₂ capture company filters on the left to select one or multiple companies.
  • Observe how the map updates to show only the projects that match your selections.
  • Review the table at the bottom: it lists the project label, company (role and share), country, and a hyperlink for more information.
  • Click the hyperlink in the table to open the external project reference in your browser.
  • Clear filters to return to a full overview of all companies and projects.
