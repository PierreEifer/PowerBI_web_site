Page 1: Map of E-Fuel Projects

This page displays an interactive world map showing all E-Fuel projects included in the dataset. Each point on the map represents a project, and its color indicates the project status (announced, operational, cancelled, etc.).

AVAILABLE FILTERS:

  • Company: Displays projects associated with a specific company.
  • Project: Allows selection of individual projects.
  • Super region: Filters projects by geographic area (Europe, Asia, Africa, etc.).
  • E-Fuel: Filters by the type of e-fuel (e-ammonia, e-methanol, e-kerosene, etc.).

HOW TO USE THIS PAGE:

  • Click on a geographic area to zoom in.
  • Use the filters on the left and right to focus the analysis on a specific company, e-fuel type, or region.
  • The total number of projects (shown in the bottom right) updates automatically based on selected filters.

Page 2: Cumulative Production of Chosen Metrics

This page visualizes the cumulative production of different metrics over time, such as CO₂ capture, e-fuel production, and hydrogen (H₂) production in ton/year.

MAIN CHART:

  • The horizontal axis (X-axis) shows the year.
  • The vertical axis (Y-axis) shows the cumulative quantity (in tons per year).
  • The blue bars illustrate how production evolves over time for the selected metric.

AVAILABLE FILTERS:

  • Metric: Choose between CO₂ capture, e-fuel production, or H₂ production.
  • E-Fuel: Filters by type of e-fuel involved.
  • Super region: Filters the data by geographic region.

HOW TO USE THIS PAGE:

  • Select the desired metric at the bottom of the page to update the chart.
  • Use the filters to analyze production trends in a specific region.
  • The “Count of projects” indicator automatically updates based on the selected filters.

Page 3: HESC (Liquified Hydrogen Energy Supply Chain)

This page summarizes the HESC project. It lets you choose one project and see its geographic context view and trace the hydrogen-to-e-fuel pathway from power source through CO₂ capture and fuel output.

AVAILABLE FILTERS:

  • Use the Project slicer on the left to select one project. This controls every other visual on the page.

HOW TO USE THIS PAGE:

  • Select the desired project in the project filter.
  • After selecting a project, click the Open Web Link – Click Here + Ctrl button. Hold Ctrl while clicking to open the external reference in your browser.
  • The map on the right shows the project’s location. Zoom/drag to confirm geographical context.
  • Power Source block shows the renewable feed (e.g., Solar).
  • Electrolyze section lists technology, vendor, installed/published/estimated capacities. Blank fields indicate missing published data; estimated values (e.g., 307 MW) provide the model’s best guess.
  • Hydrogen production boxes display published vs. estimated tonnage per year. Blank fields indicate missing published data.
  • CO₂ section captures emissions data: technology, emitter, source type capture company, and captured CO₂ volumes. Blank fields indicate missing published data.
  • E‑fx pathway & e‑fuel production show the synthetic fuel route plus published/estimated fuel tonnage per year. Blank fields indicate missing published data. Use these to assess project scale and compare them to other entries.

Data Consistency Watch

For some projects, the published values for H₂, CO₂, and E-Fuel are not consistent with each other.

Estimations are made using stoichiometric coefficients for each type of fuel: ammonia, methanol, methane, kerosene…

This explains the significant discrepancies between the estimate and the published values.
